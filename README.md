# pi-cfg

Shared pi coding agent config and bootstrap scripts.

## Quick Setup

```bash
git clone git@github.com:tsnAnh/pi-cfg.git ~/.pi
cd ~/.pi
chmod +x scripts/setup-pi.sh
./scripts/setup-pi.sh
```

Installs all packages from `settings.json` into the current project's `.pi/`.

## Packages

- **npm:** pi-hooks, pi-rtk-optimizer, pi-observational-memory, pi-mermaid, amp-themes, pi-powerline-footer, pi-simplify
- **git:** pi-codemapper, pi-minimal-subagent, pi-fork
